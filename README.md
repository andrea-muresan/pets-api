Assignment

When you feel comfortable with Spring Boot, create a GET REST endpoint that generates a list of pets with the following fields:  
name  
owner  
type (cat/dog/etc.)  
race  
realAge  
humanAge (pet years)  
Nice to have: The list should be sortable and filterable.  
Examples:  
GET /api/pets?type=dog  
GET /api/pets?sort=realAge,desc
